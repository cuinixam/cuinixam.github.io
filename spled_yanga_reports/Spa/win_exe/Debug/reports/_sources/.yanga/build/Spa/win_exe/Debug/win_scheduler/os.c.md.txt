### OsTask

```{code-block} c
:linenos:
:lineno-start: 5

static void OsTask(void)
{
    spled();
    // Delay in microseconds
    usleep(CONFIG_OS_TASK_PERIOD * 1000); // CONFIG_OS_TASK_PERIOD is in milliseconds
}
```

### OsRun

```{code-block} c
:linenos:
:lineno-start: 12

void OsRun(void)
{
    while (1)
    {
        OsTask();
    }
}
```
