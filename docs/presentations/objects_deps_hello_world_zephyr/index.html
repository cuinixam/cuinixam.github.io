Object Analyzer Help

Search

Type in the search box to find specific nodes. Click a result to zoom and center on it.

Interaction

  • Select Edge: Click an edge to highlight it and its connected (source/target) nodes.
  • Select Node: Click a node to see a red border and highlight its immediate neighbors.
  • Double-Click Node: Finds the node (or its parent folder) in the Directory Filter tree on the right, expands the tree, and highlights the folder.
  • Clear Selection: Click on the background canvas.

Directory Filter

Use the panel on the right to toggle visibility of files and folders.

  • Checkboxes: Hide/Show specific directories and their contents.
  • Buttons: Use "Expand All", "Collapse All", etc., to manage the tree view.
