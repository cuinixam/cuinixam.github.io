pypeline

Define once, run anywhere

Define your pipeline once in YAML and run it consistently across local environments and any CI/CD platform.

github.com/cuinixam/pypeline
pypeline.yaml
pipeline:
  - step: CreateVEnv
    module: pypeline.steps.venv

  - step: PreCommit
    module: pypeline.steps.precommit

  - step: PyTest
    module: pypeline.steps.pytest

  - step: BuildDocs
    module: pypeline.steps.sphinx
    config:
      source: "docs"
      output: "build/docs"

# $ pypeline run
# → identical results, every machine.
